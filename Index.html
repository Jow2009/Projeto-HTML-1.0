Surgimento da internet

(Voltar)
A Web não conecta apenas máquinas, ela conecta pessoas.

A internet surgiu no final dos anos 1960 com o projeto ARPANET, criado pelo Departamento de Defesa dos EUA para conectar computadores de centros de pesquisa.

A ideia era permitir a troca rápida de informações mesmo em caso de falhas na rede.

Nos anos 1980, universidades passaram a usar essa rede, e o protocolo TCP/IP foi adotado como padrão.

Em 1990, surgiu a WWW, criada por Tim Berners-Lee, facilitando o acesso por meio de sites e navegadores.

A partir daí, a internet se popularizou globalmente.

Como funciona o HTML

(Voltar)

O HTML funciona como a estrutura básica de uma página web, organizando o conteúdo em elementos.

Ele usa tags para marcar textos, imagens, links, títulos, etc.

O navegador lê essas tags e exibe o conteúdo de forma visual para o usuário.

Junto com o CSS e o JavaScript, ele forma a base de qualquer site.

O que é Client e Server?

(Voltar)

No mundo da tecnologia, "Client" (cliente) é o dispositivo ou programa que faz uma solicitação de informações ou serviços. Pode ser um navegador de internet, um aplicativo ou até um celular acessando um site.

Já o "Server" (servidor) é o computador ou sistema que recebe essas solicitações e fornece os dados ou serviços pedidos. Ele pode hospedar sites, bancos de dados, arquivos, entre outros.

Quando você acessa um site, seu navegador (Client) envia um pedido ao servidor onde o site está armazenado. Esse pedido segue um protocolo, geralmente o HTTP ou HTTPS.

O servidor então processa a solicitação, localiza as informações e responde com os dados, como textos, imagens ou vídeos, que serão exibidos no navegador do cliente.

Essa troca de informações acontece em segundos e de forma contínua, permitindo que páginas sejam carregadas, vídeos sejam assistidos e dados sejam enviados e recebidos.

Em resumo, o Client inicia a comunicação e o Server responde, formando juntos a base da navegação e funcionamento da internet como conhecemos.

O que são Tags?

(Voltar)

Tags são elementos da linguagem HTML usados para marcar e estruturar o conteúdo de uma página web.

Elas indicam ao navegador como exibir textos, imagens, links e outros componentes.

Cada tag geralmente vem em pares, com uma abertura (<tag>) e um fechamento (</tag>), embora algumas sejam autofecháveis.

Aqui vão algumas tags comuns:

  • <html>: Define o início do documento HTML
  • <head>: Contém metadados e links de estilo
  • <body>: Onde o conteúdo visível da página aparece
  • <h1> até <h6>: Títulos e subtítulos, do maior ao menor
  • <p>: Parágrafo de texto
  • <img>: Imagem (autofechável)
